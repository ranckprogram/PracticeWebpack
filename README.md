### 多页处理

- new htmlWebpackPlugin 多调用几次，传不同参数，引用不同的模板
- entry 定义多个 output 输出对应
- 每个页面指定chunks，或者排除chunk
- 公共js，直接插入，不用外连接，compilation.assets
